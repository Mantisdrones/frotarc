🔑 API Infosimples
⚠️ O token fica salvo apenas no seu navegador (localStorage). Nunca é enviado para outro lugar além da API da Infosimples. Crie sua conta em api.infosimples.com.
Credenciais gov.br (para o DETRAN-MG)

O DETRAN-MG exige login gov.br para consultar. Você pode definir credenciais globais ou por veículo. As credenciais ficam salvas só no seu navegador.

⚠️ A senha gov.br é transmitida diretamente para a Infosimples (que a usa para acessar o DETRAN-MG). Leia os termos de uso da Infosimples antes de usar.
Agendamento mensal

O sistema não pode executar consultas em segundo plano automaticamente (limitação de apps web). A alternativa é deixar a aba aberta no dia agendado, ou usar a consulta manual mensalmente.
